private helper function to build view

_loadviews(string $body_view, array $data) : void

This loads views for the registration, failed-login, etc. screens The following parts are sent in order:

  • include/header.php
  • header/simple.php : no login box
  • leftnav/default.php: the default content of the navigation bar
  • body/$body_view_: the body content given as a parameter
  • include/footer.php
